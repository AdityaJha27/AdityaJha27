## 🚀 Projects

### 🔍 FinTech Fraud Detection System
> `XGBoost` `Random Forest` `SHAP` `SMOTE` `Live Monitoring`

Production fraud detection platform trained on **6.3M+ financial transactions**. Tackled real-world class imbalance with SMOTE, added SHAP-based explainability at the feature level, and built live monitoring dashboards. The goal was a system a business could actually trust and deploy — not just one that scores well on a holdout set.

---

### ✈️ Flight Fare Prediction System
> `XGBoost` `Feature Engineering` `Hyperparameter Tuning` `Streamlit`

End-to-end regression pipeline for real-time fare prediction — from raw data to a deployed Streamlit app. Focused on systematic feature engineering and tuning, with production deployment as the output, not a notebook.

---

### 👗 AI Fashion Discovery — Visual Similarity Search
> `ResNet50` `Cosine Similarity` `Embedding Extraction` `Flask`

Visual search engine using ResNet50 embeddings and cosine distance for image-based product retrieval. Built a Flask pipeline to serve results at query time — same core architecture used in production-scale visual search.

---

### 🤖 AMD AI Reinforcement Learning Hackathon · IIT Delhi
> `LLM Agents` `Reinforcement Learning` `AMD MI300X GPUs`

Trained and evaluated LLM-based agents under reinforcement learning constraints on AMD MI300X hardware. Focused on agent behavior under compute limits, reward signal design, and evaluation methodology — direct experience with agentic systems under real constraints.
